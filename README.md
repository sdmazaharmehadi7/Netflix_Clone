# 🎬 Netflix Clone

A fully responsive **Netflix landing page clone** built using modern web technologies. This project replicates the look and feel of Netflix’s homepage with a focus on UI design, responsiveness, and clean layout.

## 📌 Features

* 🎨 **Modern UI Design** – Inspired by Netflix homepage
* 📱 **Fully Responsive** – Works on mobile, tablet, and desktop
* ⚡ **Fast Loading** – Optimized layout and assets
* 🧩 **Reusable Components** – Structured and clean code
* 🎥 **Hero Section** with call-to-action
* 📄 **Multiple Sections** (FAQ, Footer, etc.)

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **Flexbox & Grid** – Layout design

1. Clone the repository


3. Open `index.html` in your browser
